# Logo Update Instructions

To update the logo with the new image file:

1. Place the file `e4343ea8-1905-455a-9c6e-e9982b08e352.jpg` in the `assets` folder
2. Rename it to `logo-image.jpg` (or update all references in HTML files)
3. The logo will automatically appear on all pages

Current logo path: `assets/logo-image.jpg`

Files that reference the logo:
- index.html
- today.html
- directory.html
